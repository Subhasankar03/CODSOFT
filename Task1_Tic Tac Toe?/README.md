# Tic Tac Toe AI

## Project Overview

Tic Tac Toe AI is a simple web-based game developed using HTML, CSS, and JavaScript. It allows a user to play Tic Tac Toe against an AI opponent. The AI uses the Minimax algorithm to make optimal moves, making it difficult for the player to win.

## Features

- Play against an intelligent AI opponent.
- Interactive and responsive user interface.
- Displays the game result (Win, Lose, or Draw).
- Restart game option.
- Clean and modern design.
- Built using only HTML, CSS, and JavaScript.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
