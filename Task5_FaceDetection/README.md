# AuraVision - AI Face Detection & Recognition

## Project Overview
AuraVision is an AI-powered Face Detection and Recognition web application developed as part of the CodSoft AI Internship (Task 5). It detects human faces from a live webcam or uploaded image and displays facial landmarks with confidence scores.

## Features
- Real-time face detection using webcam
- Face recognition demo
- Facial landmark detection
- Upload image for face detection
- Live confidence score
- Modern and responsive dashboard
- AI status monitoring

## Technologies Used
- HTML5
- CSS3
- JavaScript
- face-api.js
- WebRTC (Webcam API)

## How to Run
1. Download or clone the project.
2. Open the project folder.
3. Open index.html in a web browser.
4. Allow camera permission.
5. Start detecting faces.

- index.html
- styles.css

- assets/
- models/
